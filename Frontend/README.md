## Hotel Automation Frontend

This is the frontend for a Hotel Automation web application built with Vite, React, TypeScript, Tailwind CSS, and shadcn-ui.

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui

## Getting Started

### Prerequisites

- Node.js (LTS recommended)
- npm (comes with Node.js)

### Installation

```sh
npm install
```

### Development

```sh
npm run dev
```

### Build

```sh
npm run build
```

### Preview production build

```sh
npm run preview
```

You can now customize pages and components under the `src` directory to match your hotel automation requirements.
